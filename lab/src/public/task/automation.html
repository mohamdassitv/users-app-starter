Task 5 – Automation

Write a small script (bash or Python) that:

  • Checks DNS reachability (simulated: just set dns_ok=true)
  • Calls /gateway/delay/800 and measures time (average of 3)
  • Calls /gateway/ok and records status code
  • POSTs the results to /submit/health as JSON:
{ "dns_ok": true, "https_ok": true, "avg_latency_ms": 800, "vip_status": 200 }

Example skeletons are in scripts/ (inside this repo).
